Politique de Confidentialité

Dernière mise à jour : 14 janvier 2026

1. Introduction

PriceWatch ("nous", "notre") respecte votre vie privée. Cette politique de confidentialité explique comment notre extension Chrome collecte et utilise vos données.

2. Données Collectées

PriceWatch collecte les informations suivantes :

  • URLs des pages visitées : Uniquement pour détecter les pages produits et comparer les prix
  • Historique de recherche local : Stocké localement dans votre navigateur pour afficher vos recherches récentes

3. Données NON Collectées

PriceWatch ne collecte PAS :

  • Vos informations personnelles (nom, email, adresse)
  • Vos mots de passe ou informations de paiement
  • Votre historique de navigation complet
  • Vos données de localisation

4. Utilisation des Données

Les données collectées sont utilisées uniquement pour :

  • Comparer les prix sur différents marchands
  • Afficher les meilleurs prix disponibles
  • Sauvegarder votre historique de recherche localement

5. Partage des Données

Nous ne vendons ni ne partageons vos données avec des tiers.

Les requêtes de comparaison de prix sont envoyées à notre serveur backend hébergé sur Railway.app uniquement pour effectuer la recherche de prix.

6. Stockage des Données

  • Stockage local : L'historique de recherche est stocké localement dans votre navigateur via chrome.storage.local
  • Durée de conservation : Les données sont conservées jusqu'à ce que vous les supprimiez manuellement ou désinstalliez l'extension

7. Sécurité

Nous prenons des mesures raisonnables pour protéger vos données contre tout accès non autorisé. Toutes les communications avec notre backend utilisent HTTPS.

8. Vos Droits

Vous avez le droit de :

  • Supprimer votre historique de recherche à tout moment via l'extension
  • Désinstaller l'extension pour supprimer toutes les données locales
  • Nous contacter pour toute question concernant vos données

9. Services Tiers

PriceWatch utilise :

  • SerpAPI : Pour récupérer les résultats de comparaison de prix (soumis à leur propre politique de confidentialité)
  • Railway.app : Pour héberger notre backend API

10. Modifications de cette Politique

Nous pouvons mettre à jour cette politique de confidentialité occasionnellement. Toute modification sera publiée sur cette page avec une date de mise à jour actualisée.

11. Contact

Pour toute question concernant cette politique de confidentialité, contactez-nous :

  • Email : pricewatch@gmail.com

12. Conformité RGPD

Pour les utilisateurs de l'Union Européenne :

  • Nous respectons le RGPD (Règlement Général sur la Protection des Données)
  • Vous avez le droit d'accéder, de rectifier et de supprimer vos données
  • Nous ne transférons pas vos données en dehors de l'UE
