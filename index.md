# Politique de Confidentialité - PriceWatch

**Dernière mise à jour : 10 janvier 2026**

## Introduction

PriceWatch ("nous", "notre", "l'extension") s'engage à protéger votre vie privée. Cette politique explique quelles données nous collectons et comment nous les utilisons.

## Données Collectées

### Ce que nous NE collectons PAS :
- ❌ Aucune donnée personnelle (nom, email, adresse)
- ❌ Aucun historique de navigation
- ❌ Aucune information de paiement
- ❌ Aucun mot de passe
- ❌ Aucune donnée sensible

### Ce que nous collectons :
- ✅ **Requêtes de comparaison anonymes** : Lorsque vous comparez un prix, nous envoyons le nom du produit à notre serveur pour récupérer les prix. Cette requête est anonyme et ne contient aucune information personnelle.
- ✅ **Cache local** : Les résultats de comparaison sont sauvegardés temporairement dans votre navigateur (pas sur nos serveurs) pour améliorer la vitesse.

## Utilisation des Données

Les requêtes de comparaison sont utilisées uniquement pour :
1. Rechercher les prix sur différents sites marchands
2. Vous afficher les résultats
3. Améliorer la qualité du service

**Nous ne vendons, ne partageons, ni ne transférons vos données à des tiers.**

## Services Tiers

### SerpAPI (Google Shopping)
Pour récupérer les prix, nous utilisons SerpAPI qui interroge Google Shopping. Les données envoyées sont :
- Nom du produit uniquement
- Aucune donnée personnelle

Politique de confidentialité de SerpAPI : https://serpapi.com/privacy

### Liens d'affiliation
PriceWatch peut contenir des liens d'affiliation vers des sites marchands (Amazon, Fnac, etc.). Lorsque vous cliquez sur un lien et effectuez un achat, nous pouvons recevoir une commission. Cela ne change en rien le prix pour vous.

## Stockage des Données

- **Sur votre appareil** : Les résultats de comparaison sont stockés temporairement dans le cache de votre navigateur Chrome (Storage API). Vous pouvez les effacer à tout moment.
- **Sur nos serveurs** : Nous stockons temporairement (1 heure) les résultats de comparaison pour améliorer les performances. Aucune donnée personnelle n'est associée.

## Cookies

PriceWatch n'utilise pas de cookies de tracking ou publicitaires.

## Permissions Chrome

PriceWatch demande les permissions suivantes :
- **activeTab** : Pour détecter les pages produits et afficher le bouton de comparaison
- **storage** : Pour sauvegarder les résultats en cache local
- **host_permissions (https://*/*)** : Pour fonctionner sur tous les sites e-commerce

Ces permissions sont nécessaires au fonctionnement de l'extension et ne sont jamais utilisées à d'autres fins.

## Sécurité

Nous prenons la sécurité au sérieux :
- Toutes les communications avec notre serveur sont chiffrées (HTTPS)
- Aucune donnée sensible n'est transmise ou stockée
- Le code source est régulièrement audité

## Vos Droits

Vous avez le droit de :
- Désinstaller l'extension à tout moment
- Effacer le cache local via les paramètres Chrome
- Nous contacter pour toute question : [pricewatch@gmail.com]

## Modifications de cette Politique

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications seront publiées sur cette page avec une date de mise à jour.
